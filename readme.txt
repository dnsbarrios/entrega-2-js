# Entrega Final - Denis Barrios

## Proyecto: Tienda Deportiva Éxodo Sport

### Descripción
Este simulador web representa una tienda de ropa y artículos deportivos orientada a los usuarios de Éxodo Academia. A través de la interfaz los usuarios pueden seleccionar productos, agregarlos al carrito, calcular el total de la compra (con impuestos) y gestionar sus compras desde el navegador.

---
### Funcionalidades implementadas

✅ Productos cargados dinámicamente con `fetch()`  
✅ Carrito de compras funcional con almacenamiento en `localStorage`  
✅ Cálculo de precio final con IVA (22%)  
✅ Botones para eliminar productos o vaciar todo el carrito  
✅ Permite simular la compra ofreciendo cuotas y calculando intereses si corresponde.
✅ Interacción completamente desde el DOM
✅ Notificaciones visuales con librería externa


---

### Observaciones finales

Este trabajo simula una tienda digital de manera profesional, utilizando buenas prácticas de estructura, legibilidad y escalabilidad en el código. La estética visual está inspirada en la marca **Éxodo Academy**, y el simulador está preparado para integrarse en futuras etapas del proyecto completo.

Esta entrega cierra el ciclo del curso de Desarrollo Web, demostrando el uso completo de:
- HTML + CSS
- JavaScript + DOM
- Funciones reutilizables
